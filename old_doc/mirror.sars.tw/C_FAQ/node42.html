4.6 可是 && 和 || 運算符呢？ 我看到過類似  while((c = getchar()) != EOF && c != '\n')  的代碼 ……

這些運算符在此處有一個特殊的 ``短路" 例外: 如果左邊的子表達式決定最終 結果 (即,真對於 || 和假對於 && ) 則右邊 的子表達式不會計算。因此, 從左至右的計算可以確保, 對逗號表達式也是如此。而且, 所有這些運算符  (包括 ? : ) 都會引入一個額外的內部序列點 (參見問題 3.7)。

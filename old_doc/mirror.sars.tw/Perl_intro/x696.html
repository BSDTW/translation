5.8 三元運算子

三元運算子是程式老手最愛用的一種流程控制寫法。

#! /usr/bin/perl

chomp($i=<STDIN>);

$rt = ($i == 10) ? 'yes' : 'no'; # 若 $i 等於 10，則 $rt 為 yes，否則 $rt 為 no

print "$rt\n";

上式其實可以用 if 來改寫：

#! /usr/bin/perl

chomp($i=<STDIN>);

if ($i == 10) {
	print "yes\n";
} else {
	print "no\n";
}

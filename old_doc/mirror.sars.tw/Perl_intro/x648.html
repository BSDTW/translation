5.2 unless 控制結構

if 是判斷條件為真時做某事，unless 剛好相反：unless 是判斷條件為假時做某事。

unless 有三種形式：

第 1 種：

unless (條件判斷為假) {
	做某些事情 ....
}


第 2 種：

unless (條件判斷為假) {
	做某些事情 ....
} else {
	做另一些事情 ....
}

第 3 種：(後置式)

做某事 unless (條件判斷為假);

例：

print "I do like it !\n" unless ($yourANS eq '我不喜歡輸的感覺');

用例

# 除非 192.168.1.2 的主機名稱存在，否則：

unless (exists $HOSTNAME{'192.168.1.2'}) {
   print "主機名稱不存在!\n";
} else {
   print  "主機名稱為 $HOSTNAME{'192.168.1.2'}";
}

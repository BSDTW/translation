:first-line版本：CSS2　 兼容性：IE5.5+
語法：
Selector : first-line { sRules }
說明：
設置對像內的第一行的樣式。
此偽對像僅作用於塊對象。內聯要素要使用該屬性，必須先設定對象的 height 或 width 屬性，或者設定 position 屬性為 absolute ，或者設定 display 屬性為 block 。
如果未強制指定對象的 width 屬性， 首行的內容長度可能不是固定的。
示例：
p a:first-line { color: green }
div:first-line { color:red;font-size:16px; }
